# Свадебное приглашение (HTML/CSS/JS)

Простой статичный сайт без сборки. Открывается напрямую в браузере.

## Как запустить
1. Распакуйте архив.
2. Дважды кликните `index.html` — откроется в браузере.
   Или запустите локальный сервер:
   ```
   python3 -m http.server 8000
   ```
   и откройте http://localhost:8000

## Как редактировать в VSCode

### Свадебные данные (имена, дата, телефон и т.д.)
Откройте **`index.html`** — весь текст прямо в разметке (ищите имена «Бурхониддин», «Мухибахон», дату и т.д.).

Дата обратного отсчёта и текст для «Поделиться» — в **`script.js`** (объект `WEDDING` вверху файла).

### Цвета
Файл **`styles.css`** — блок `:root` в самом верху. Меняйте переменные:
- `--accent`, `--gold` — золотые акценты
- `--ink` — тёмный текст
- `--cream`, `--background` — фоны

### Музыка
Замените файл `assets/wedding-music.mp3` на свой (сохраните то же имя).

### Фото и веточка
Замените `assets/floral-hero.jpg` и `assets/sprig.png` на свои.

## Публикация
Загрузите всю папку на любой хостинг статики: GitHub Pages, Netlify, Vercel, Cloudflare Pages — просто перетащите папку.
"# wedding-invitation"
